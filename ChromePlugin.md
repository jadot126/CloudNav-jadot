# 浏览器插件更新

src\components\SettingsModal.tsx

## 已完成功能

### 书签同步功能

在浏览器扩展的侧边栏中新增了书签同步功能：

#### 1. 上传浏览器书签到网站
- 获取当前浏览器所有书签
- 可以选择上传到指定文件夹（如 /chrome）
- 支持两种同步模式：
  - **自动合并**：保留现有数据，跳过重复项
  - **强制覆盖**：清空目标文件夹及其所有子目录后重新导入
- 上传前显示二次确认弹窗，显示书签数量和同步模式
- 自动保留浏览器书签的文件夹结构

#### 2. 同步网站书签到浏览器
- 将指定分类下的所有书签同步到浏览器书签栏
- 支持两种同步模式（同上传功能）：
  - **自动合并**：保留现有书签，跳过重复项
  - **强制覆盖**：清空浏览器中的目标文件夹后重新导入
- 同步前显示二次确认弹窗
- 自动创建对应的文件夹结构

#### 使用说明
1. 下载并安装浏览器扩展（从设置页面的"扩展工具"标签页下载）
2. 打开扩展侧边栏（快捷键 Ctrl+Shift+E）
3. 点击搜索框旁边的上传图标按钮，展开书签同步面板
4. 在"目标文件夹名称"输入框中设置文件夹名称（默认为 chrome）
5. 选择同步模式（自动合并或强制覆盖）
6. 点击对应按钮执行操作：
   - **上传到网站**：将浏览器书签上传到 CloudNav
   - **同步到浏览器**：将 CloudNav 书签同步到浏览器

#### 技术实现
- 使用 Chrome Bookmarks API 进行书签读写操作
- 支持递归处理嵌套的书签文件夹结构
- 通过 URL 去重避免重复导入
- 实时显示同步进度
- 所有功能集成在插件的 sidebar.js 中，用户下载插件后即可使用

#### 权限说明
插件需要以下权限：
- `bookmarks`：读写浏览器书签
- `storage`：本地数据缓存
- `activeTab`、`scripting`：页面交互
- `sidePanel`：侧边栏功能
- `contextMenus`：右键菜单
- `notifications`：通知提示
