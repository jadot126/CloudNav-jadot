# 更新导航页项目

- 参考 reference 项目，更新导航页项目
- 项目说明
  - 部署：使用 Cloudflare Pages 部署
  - 数据存储：使用 Cloudflare KV 数据，参考 functions 中 api 的使用
  - functions: 保留结构，用于部署到 Cloudflare 作为应用 api
- 更新说明
  - **重要：** 保留原项目样式，布局结构
  - **重要：** 保留原有基本功能
  - 优化前端页码代码结构

- 功能更新  
  - 环境变量配置参考： reference/functions/api/link.ts 中 interface Env
  - 环境变量：仅保留 CLOUDNAV_KV，用于访问 Cloudflare KV 数据库
  - 初始化：初次登录时读取 KV 数据验证是否设置管理员密码，未设置提示设置后才能访问
  - 管理员登录后功能权限：增加，删除，修改 标签页，标签页分组
  - 标签页分组访问：每个分组作为 /uri 来访问，增加分组或修改分组时支持修改访问 uri，支持增加子分组
  - 标签页分组访问验证：增加时默认无密码验证，管理员可以分别设置密码验证，未验证访问分组 uri 时提示身份验证，父目录权限密码可以管理所有子目录分组
